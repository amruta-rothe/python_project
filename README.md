College Lost & Found System

A desktop-based Lost & Found Management System developed using Python Tkinter.
The application helps colleges manage lost and found items digitally with a simple and efficient interface.

Overview

This system replaces traditional manual registers with a digital solution.
It allows users to store, search, and manage lost and found items in an organized manner.

Objectives
Digitize lost and found record management
Reduce manual effort and errors
Enable quick search and tracking
Maintain structured records with timestamps
Provide export functionality for reports
User Roles
The system mainly includes an Admin user who can add, update, delete, and manage all records. Only authorized users can access the system through login authentication

Admin

Add, update, and delete items
Export records to CSV
View system statistics

Student

View available items
Search and filter records
Key Features
Login system for admin and student
CRUD operations (Add, Update, Delete items)
Unique item ID generation
Automatic date and time stamping
Search and filter functionality
CSV export support
Dashboard with statistics
Dark mode toggle
Image upload support
JSON-based data storage
Tech Stack
Language: Python
GUI: Tkinter (ttk)
Storage: JSON
Concepts: Object-Oriented Programming
